# 🎨 Color Psychology Cheatsheet

A high-performance, visual reference tool designed for developers and UI/UX designers to understand the emotional and psychological impact of colors in digital environments.

![Project Preview](/placeholder.png) 

## 🚀 Overview

The **Color Psychology Cheatsheet** is a curated interactive guide that breaks down the meaning behind common UI colors. Built with a "dark mode first" approach, it utilizes high-contrast accents to demonstrate how color can drive user behavior, foster trust, and project technological power.

This project was built using **AI-driven development (Vibecoding)**, focusing on rapid iteration, clean aesthetics, and functional minimalism.

## 🧠 Core Concept: Why Color Matters?

In modern UI/UX, color is a silent language. This tool provides:
- **Psychological Insights**: Instant access to the emotions each color triggers.
- **Visual Hierarchy**: Demonstrates how to use "Neon" and "Cyber" tones to direct attention.
- **Accessibility**: Optimized for dark interfaces to reduce eye strain while maintaining vibrancy.

## ✨ Key Features

* **Interactive Drawer**: A sleek, click-to-toggle side panel containing project philosophy and design strategy.
* **Vibecoding Aesthetic**: Modern "Glassmorphism" effects, high-contrast borders, and neon accents.
* **Quick Reference Cards**: Organized layout for rapid consultation during the design process.
* **Responsive Design**: Smooth transitions and layouts that adapt to any screen size.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Backdrop-filters (glassmorphism), and Flexbox/Grid.
- **JavaScript**: Minimalist logic for the interactive drawer.
- **Vibecoding Workflow**: Leveraging AI to accelerate design and code implementation.

## 📁 Structure

- `index.html`: Main interface and content.
- `style.css`: The "Cyber-Power" theme and drawer animations.
- `script.js`: Interactive logic for the sliding side-panel.
